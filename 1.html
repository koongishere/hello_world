링크를 걸고싶은 것에 a태그를 감싼다. 여기서 속성을 빼면 안된다. hypertext는 링크와 동의어고 ref는 reference의 약자래. 그래서 href라는 연결될 링크를 기재할 속성태그를 기입한다. 새탭을 통해 링크를 연결하고, 마우스를 갖다대면 툴팁이 나타나도록 하는 속성태그도 a태그와 함께할수있다.

스타일 태그를 쓴다 스타일이라는 html의 속성을 쓴다 검색추천어 CSS TEXT SIZE PROPERTY CSS TEXT CENTER PROPERTY CSS SELECTORS property를 모두 외우지 않아도 된다. 우리는 검색하는 방법, 이런것들이 있다는것만 알면돼. 효과 프로퍼티 선택자selector에 대해 알아보자이제 오 선택자 어려움. 상위의 명령이 우선적으로 실행됨 원리를 이해해보자. 왜 태그 선택자와 클래스 선택자, 아이디 선택자. 아이디-클래스-태그 순으로 강함. 아이디가 가장 강함. 아이디 값은 단 한번만 등장해야함.
